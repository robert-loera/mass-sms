# Mass SMS Program

**Version 1.0.0**

## Description
This program allows users to connect their twilio account and send unique messages
to different users by just uploading a csv file.

## Installation
- Install packages with pip: ```pip install -r requirements.txt```


## Implementation
![](https://github.com/robert-loera/mass-sms/blob/main/gui-ss.png)
- Select a file you would like to send messages to with the choose file button
- Type your message in the message box
- Press send when finished and all messages will be sent and return a status
